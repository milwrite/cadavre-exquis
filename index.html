Exquisite Corpse

“The exquisite corpse will drink the new wine.” the game’s first sentence — Paris, 1925

MoMA defines it as a game in which each participant takes turns writing or drawing on a sheet of paper, folding it to conceal the contribution, and then passing it to the next player for a further contribution.

This version seats a language model at the table. Each player adds a few words, and the sheet is folded as it passes — only the newest contribution stays visible as the next player’s cue. When someone ends the game, the sheet unfolds.
